# espanso-package-greek-letters
Easy Greek letters using the [Espanso](https://espanso.org/) text expander. 

<kbd>compose</kbd> got you frustrated? Sick of changing your keyboard layout to type a single Greek character? Install [Espanso](https://espanso.org/), install this package, and you can type out Îž with <kbd>:gX</kbd>. Just that simple.

For more information, read the [documentation](https://espanso.org/docs/)
